# SMALLTALK
Stupid simple chat server in 90 lines of Java with no external dependencies, coded in a hot summer afternoon after watching [Antirez's Smallchat videos](https://www.youtube.com/watch?v=eT02gzeLmF0).
## Usage
Build with `mvn clean package`, then run with `java -jar target/smalltalk-1.0-SNAPSHOT.jar`, then connect via telnet or netcat at port 1234.
